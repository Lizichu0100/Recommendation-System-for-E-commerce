Bước 1: Mở cmd quyền admin và tải streamlit: pip install streamlit, nếu đã tải rồi thì bỏ qua bước 1.
Bước 2: Cũng trong cmd, nhập "cd <đường dẫn thư mục>": Trong đó <đường dẫn thư mục> phải chứa hai file csv là purchase_history.csv & recommendations.csv + file Demo_web.py
Bước 3: Nhập "streamlit run Demo_web.py"
